book structure · 書籍構成

書籍のアーキテクチャ The book's architecture
read through the C4 Model

書籍そのものをC4モデルの4層構造で設計。第1部（コンテキスト）→第2部（コンテナ）→第3部（コンポーネント）→第4部（実践）へと階層的に深化する。 The book itself is designed in the C4 Model's 4-layer structure, deepening hierarchically from Part 1 (Context) → Part 2 (Container) → Part 3 (Component) → Part 4 (Practice).

L1 · CONTEXT — 第1部
俳句の世界へようこそ
俳句とは何か・C4モデルとは何か・この本の使い方。動画を入口に、初心者・高齢者が安心して入れる序章。
第1章：まずは動画で味わう 第2章：俳句工学の考え方 第3章：C4モデルとは
L2 · CONTAINER — 第2部
芭蕉の名句を工学的に分解する
古池や・夏草や・閑さや の3句をC4モデルL1〜L3で完全分解。各句ごとに統一フォーマット（動画QR→C4図→認知プロセス→実践ワーク）で解説。
古池や（音の設計） 夏草や（時間の余白） 閑さや（対比の工学）
L3 · COMPONENT — 第3部
C4モデル応用事例研究
芭蕉の旅路のC4モデル化・一茶と子規の比較モデル・季語辞書のコンポーネント化・読者感情のシーケンス図化。専門的なのに分かりやすい「研究成果の翻訳」章。
旅路モデル 俳人比較 感情シーケンス 俳句生成アルゴリズム
L4 · CODE — 第4部
あなたも作れる "工学的俳句"
散歩しながら音を探す・季語を変えてみる・C4テンプレートで俳句を設計する。健康×俳句×工学が自然に融合する実践ワーク集。
